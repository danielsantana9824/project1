NFL Watch

A web application for NFL fans

Project Overview

  • View team information and live scores
  • User authentication
  • Team listings
  • Detailed team information

Technologies Used

  • HTML5, CSS, JavaScript
  • Local Storage for user authentication
  • NFL API for team and score data

Key Features

  • User Registration and Login
  • NFL Team Listings with logos
  • Live Scores (when available)
  • Detailed Team Information
  • Responsive Design

Challenges Faced

  • Integrating multiple API endpoints
  • Handling asynchronous data fetching
  • Implementing responsive design

Future Enhancements

  • Add more detailed player statistics
  • Implement user favorites feature
  • Integrate with social media for sharing
